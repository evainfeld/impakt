Rustic Audio

An open source Rust DSP library, command-line tool and UI for cleaning voice recordings and encoding them as highly compressed Opus files. Created for settings where bandwidth is scarce—such as privacy-preserving communication—while preserving legibility and quality. Created with the support of NLnet foundation.
